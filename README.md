# MallScape – The Ultimate Retail Destination Experience

MallScape is a premium interactive web experience designed to showcase a mega shopping destination as a high-impact business platform. It uses cinematic visuals, modern UI, and storytelling to create an engaging digital sales experience for tenants, sponsors, and event partners.

## Features

- Fullscreen cinematic hero with autoplay video storytelling
- Non-linear floating navigation for deck-style exploration
- Animated KPI counters and luxury presentation sections
- Cinematic attraction panels for the Aquarium and VR Park
- Responsive premium UI built for desktop and tablet
- Integrated contact actions for leasing, sponsorships, and events

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

```powershell
npm.cmd install
npm.cmd run dev
```

## Build

```powershell
npm.cmd run build
```

## Project Goal

The experience is designed to emotionally engage visitors within the first 10 seconds and position the destination as the ideal platform for leasing, partnerships, and events.
